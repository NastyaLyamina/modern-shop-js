# E-commerce Store (JS Vanilla Project)

Каталог интернет-магазина с динамической отрисовкой каталога и корзиной.

[https://nastyalyamina.github.io/modern-shop-js/]

## Технические особенности
* **Модульная архитектура:** Код разделен на независимые модули (Cart, Catalog, Filters, Form Validation, UI).
* **Async Data Loading:** Загрузка данных о товарах из локального JSON-файла через `Fetch API`.
* **Dynamic Filtering & Sorting:** Реализована система фильтрации товаров (по категориям/параметрам) и сортировка по цене в реальном времени.
* **Complex UI Components:** Интеграция слайдера товаров дня и многоуровневого меню.

## Стек технологий
* JavaScript (ES6+ Modules)
* HTML5 / SCSS (BEM)
* Fetch API / JSON
* Библиотеки (Swiper, Just-validate, Tippy)
